# ⭐ Overview of the Project
The Productivity Tracker is a command - line based application designed to help users manage and organize their tasks efficiently. It allows users to add, remove, and update tasks, set deadline, categorize, and monitor their progress.

The system provides a simple and interactive interface using Pythonm, making it easy for user to track their productivity and stay organized in their daily activities.

# ⭐ Features

## Add Tasks 
- Create new tasks and assign them to a category.

## Remove Tasks
- Delete task from the list using their task number.

## Task Categories
- Modify task detials with the following options:
    - #### Update Task Name
    - #### Update Category
    - #### Set or Update Deadline
    - #### Mark task as completed

## Task Categories
- Organize tasks into:
    - #### Quiz
    - #### Assignment
    - #### Assesment Task
    - #### Notes

## Task Completion Tracking
- Tasks are marked as completed or not completed using symbols.

## Progress Bar
- Displays the percentage of completed tasks

## Interactive CLI Inteface 
- Uses Rich and Cutie libraries for a clean and user - friendly terminal experience

## Task Priority Sort
- Sorts tasks priority by their due dates.
